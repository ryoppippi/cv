# ryoppippi's cv

This repository contains my CV/Resume in PDF format, automatically built and deployed to [cv.ryoppippi.com](https://cv.ryoppippi.com).

## Prerequisites

- [Nix](https://nixos.org/) with flakes enabled

## Development

Watch for changes and recompile automatically:

```bash
nix develop -c typst watch ./ryotaro_kimura.typ
```

## Build

Build the PDF (output in `result/`):

```bash
nix build
```

## Update GitHub Stars

Update star counts for featured projects:

```bash
GITHUB_TOKEN=your_token ./scripts/update-stars.ts
```

## Deploy

Deploy to Cloudflare Workers:

```bash
nix build && nix run .#deploy
```

Preview deployment:

```bash
nix build && nix run .#preview
```

## Project Structure

```
.
├── ryotaro_kimura.typ    # Main CV source file
├── alta-typst.typ        # Typography template
├── icons/                # SVG icons
├── scripts/              # Scripts (with nix shebang)
│   └── update-stars.ts   # Update GitHub star counts
├── redirects.toml        # Redirect rules (generates _redirects)
├── flake.nix             # Nix flake configuration
└── result/               # Build output (generated by nix build)
```

## How It Works

- **Typst** compiles the CV to PDF
- **IBM Plex Sans** font is provided via nixpkgs
- **Nix** manages all dependencies and builds reproducibly
- **Wrangler** (from [emrldnix/wrangler](https://github.com/emrldnix/wrangler)) deploys to Cloudflare Workers
- **Redirects** are generated from `redirects.toml` during build

## Deployment

The CV is automatically deployed to Cloudflare Workers. All URLs redirect to the PDF file for immediate access.
