# markdown-cv

Fork from [elipapa/markdown-cv](https://github.com/elipapa/markdown-cv).  
Modified to using [astro](https://astro.build) instead of [jekyll](https://jekyllrb.com/).

## Create your own CV

Simply fork this repository  
and edit `src/pages/index.md`.

## Build

```sh
npm i
npm run build
```
